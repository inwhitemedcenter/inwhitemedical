Редагування сайту

Оберіть вкладку вище — відкриються тільки поля відповідного розділу.

Одна послуга = один рядок. Формат: емодзі | назва | опис | мітка | сторінка | visible або hidden. Сторінка: psychology, services, prices, booking або повне https-посилання.
Одна кнопка = один рядок. Формат: емодзі | назва | visible або hidden. Приклад: 💉 | Вакцинація | hidden
Можна вставити емодзі, посилання на фото, шлях assets/photo.jpg або обрати файл з комп'ютера. Для швидкого сайту краще фото до 2 МБ.
Один пункт = один рядок.
Один пункт = один рядок. Формат: ярлик/емодзі | назва | опис | підпис | visible або hidden.
Один відгук = один рядок. Формат: зірки | текст | ім'я | деталі | visible або hidden. Приклад: 5 | Дуже вдячна за підтримку. | Марія К. | Онлайн-консультація | visible
Один пункт = один рядок. Формат: питання | відповідь | visible або hidden.
Один значок = один рядок. Формат: ярлик/емодзі | назва кнопки | приховане посилання | visible або hidden. На сайті показується тільки ярлик кнопки, саме посилання не виводиться текстом.
Один рядок = одна сторінка. Формат: сторінка | assets/photo.jpg або https://... Сторінки: psychology, services, team, prices, about, training.
Зберігає верхню картинку, заголовки, місію та цінності саме для розділу “Про нас”.
Новий абзац = новий рядок.
Один пункт = один рядок. Формат: емодзі/назва | опис | visible або hidden.

Supabase база даних

Зберігайте зміни сайту в базі даних, щоб вони були видимі всім користувачам. У браузер вводиться тільки URL, publishable key та логін адміністратора Supabase Auth.

Secret key і пароль PostgreSQL не вставляйте в сайт. Для запису змін створіть адміністратора в Supabase Auth і увійдіть тут через email та пароль.

Команда

Додавайте, приховуйте або змінюйте фахівців без редагування коду.

Один фахівець = один рядок. Формат: фото або емодзі | ПІБ | посада | опис | visible або hidden. Для фото вставте URL або шлях, наприклад assets/doctor.jpg. Щоб прибрати фахівця з сайту, напишіть hidden.

Онлайн-заходи мають запис, тест і сертифікат. Офлайн-заходи ведуть на ваше посилання Google Forms.

Заходи
